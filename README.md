# Save Planet Game

A cyberpunk-themed planet repair game built with React, TypeScript, Tailwind CSS, and shadcn/ui components. In this game, you play as a planet defender trying to repair damaged buildings by answering computer-related questions.

## Features

- **Cyberpunk Aesthetic**: Modern cyberpunk building assets with animated effects like smoke, fire, and fog
- **Interactive Buildings**: Click on damaged buildings to repair them
- **Question System**: Answer computer-related questions to repair buildings
- **Progress Tracking**: Monitor your repair progress with a progress bar
- **Game Timer**: Challenge yourself to repair all buildings in the shortest time
- **Responsive Design**: Works on different screen sizes
- **Beautiful UI**: Utilizes shadcn/ui components for a polished interface

## Technologies Used

- React 19
- TypeScript
- Tailwind CSS v4
- shadcn/ui components
- Vite
